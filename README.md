## 🛠 주요 기능 계획
* **주식 거래 일지 기록**: 종목명, 매수/매도 구분, 가격, 수량, 거래 시간 기록
* **인터랙티브 타임라인 차트**:
  * 주식 클릭 시 해당 종목의 매매 내역이 가로형 타임라인에 점(Dot) 형태로 표시
  * 마우스 휠 또는 드래그를 이용해 타임라인을 확대/축소하여 세밀한 시간 단위 확인
  * 확대 시 상세 단가, 수량, 체결 시간 등의 툴팁 또는 추가 정보 노출
* **포트폴리오 대시보드**: 전체 자산 요약, 보유 종목 목록 및 평단가 정보 제공

## 🚀 시작하기 (설치 및 실행 방법)

1. **의존성 패키지 설치**
   ```bash
   npm install
   ```

2. **로컬 개발 서버 실행**
   ```bash
   npm run dev
   ```

3. **프로덕션 번들 빌드**
   ```bash
   npm run build
   ```
